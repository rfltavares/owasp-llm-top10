# LLM02: Insecure Output Handling

## 🎯 Overview

**Risk Level:** High  
**OWASP Rank:** #2  
**CWE Mapping:** CWE-79 (Cross-site Scripting), CWE-94 (Code Injection)

Insecure Output Handling occurs when an LLM's output is accepted and used by downstream systems without proper validation, sanitization, or handling. This can lead to XSS, CSRF, SSRF, privilege escalation, or remote code execution vulnerabilities.

## 📋 Description

LLM outputs are often treated as trusted content and passed directly to other systems, applications, or functions without adequate security controls. This creates a significant attack surface where malicious content generated by the LLM can exploit vulnerabilities in downstream components.

### Key Characteristics

**1. Insufficient Output Validation**
- LLM outputs not validated before use
- Assumption that AI-generated content is safe
- Lack of content filtering and sanitization

**2. Direct System Integration**
- LLM outputs directly executed as code
- Outputs used in system commands
- Integration with databases and APIs without validation

**3. Cross-System Propagation**
- Malicious outputs affecting multiple systems
- Chain reactions through interconnected services
- Amplification of security vulnerabilities

## 🔍 Technical Details

### Attack Mechanisms

**1. Cross-Site Scripting (XSS)**
```javascript
// LLM generates malicious JavaScript
LLM Output: "Here's a helpful script: <script>alert('XSS');</script>"

// Application directly renders output
document.innerHTML = llmResponse; // Vulnerable!
```

**2. SQL Injection via LLM**
```sql
-- LLM generates malicious SQL
LLM Output: "'; DROP TABLE users; --"

-- Application uses output in query
query = f"SELECT * FROM products WHERE name = '{llm_output}'" // Vulnerable!
```

**3. Command Injection**
```bash
# LLM generates malicious commands
LLM Output: "file.txt; rm -rf /"

# Application executes output
os.system(f"cat {llm_output}") # Vulnerable!
```

**4. Server-Side Request Forgery (SSRF)**
```python
# LLM generates malicious URL
LLM Output: "http://internal-server:8080/admin/delete-all"

# Application makes request to LLM-provided URL
response = requests.get(llm_output) # Vulnerable!
```

### Common Vulnerability Patterns

**1. Direct Code Execution**
```python
# Dangerous: Executing LLM output as code
def execute_llm_code(llm_response):
    # LLM generates: "import os; os.system('rm -rf /')"
    exec(llm_response)  # NEVER DO THIS!
```

**2. Template Injection**
```python
# Dangerous: Using LLM output in templates
template = f"Hello {llm_output}!"
# LLM output: "{{config.SECRET_KEY}}"
# Results in template injection
```

**3. File Path Manipulation**
```python
# Dangerous: Using LLM output for file operations
def read_file(llm_suggested_path):
    # LLM output: "../../../etc/passwd"
    with open(llm_suggested_path, 'r') as f:  # Path traversal!
        return f.read()
```

**4. API Parameter Injection**
```python
# Dangerous: Using LLM output in API calls
def make_api_call(llm_params):
    # LLM output: "user=admin&action=delete_all"
    url = f"https://api.example.com/action?{llm_params}"
    return requests.get(url)  # Unvalidated parameters!
```

```python
import ast
import re

class InsecureOutputDetector(ast.NodeVisitor):
    def __init__(self):
        self.vulnerabilities = []
    
    def visit_Call(self, node):
        """Detect dangerous function calls with LLM outputs"""
        
        # Check for exec() with variables
        if (isinstance(node.func, ast.Name) and 
            node.func.id == 'exec' and 
            len(node.args) > 0):
            self.vulnerabilities.append({
                'type': 'code_execution',
                'line': node.lineno,
                'function': 'exec',
                'severity': 'CRITICAL'
            })
        
        # Check for eval() with variables
        if (isinstance(node.func, ast.Name) and 
            node.func.id == 'eval' and 
            len(node.args) > 0):
            self.vulnerabilities.append({
                'type': 'code_execution',
                'line': node.lineno,
                'function': 'eval',
                'severity': 'CRITICAL'
            })
        
        # Check for os.system() calls
        if (isinstance(node.func, ast.Attribute) and
            isinstance(node.func.value, ast.Name) and
            node.func.value.id == 'os' and
            node.func.attr == 'system'):
            self.vulnerabilities.append({
                'type': 'command_injection',
                'line': node.lineno,
                'function': 'os.system',
                'severity': 'HIGH'
            })
        
        self.generic_visit(node)
    
    def visit_JoinedStr(self, node):
        """Detect f-string usage that might be vulnerable"""
        # Check for f-strings that might contain user input
        for value in node.values:
            if isinstance(value, ast.FormattedValue):
                self.vulnerabilities.append({
                    'type': 'potential_injection',
                    'line': node.lineno,
                    'function': 'f-string',
                    'severity': 'MEDIUM'
                })
        
        self.generic_visit(node)

# Usage
def analyze_code(source_code):

    detector = InsecureOutputDetector()
    detector.visit(tree)
    return detector.vulnerabilities
```

```python
class OutputSecurityMonitor:
    def __init__(self):
        self.dangerous_patterns = {
            'xss': [
                r'<script[^>]*>.*?</script>',
                r'javascript:',
                r'on\w+\s*=',
                r'<iframe[^>]*>',
                r'<object[^>]*>',
                r'<embed[^>]*>'
            ],
            'sql_injection': [
                r"';\s*(drop|delete|update|insert|create|alter)",
                r'union\s+select',
                r'--\s*$',
                r'/\*.*?\*/',
                r';\s*shutdown',
                r';\s*exec'
            ],
            'command_injection': [
                r';\s*(rm|del|format|shutdown)',
                r'\|\s*(nc|netcat|wget|curl)',
                r'`[^`]*`',
                r'\$\([^)]*\)',
                r'&&\s*(rm|del)',
                r'\|\|\s*(rm|del)'
            ],
            'path_traversal': [
                r'\.\./.*\.\.',
                r'\.\.\\.*\.\.',
                r'/etc/passwd',
                r'\\windows\\system32',
                r'\.\..*config',
                r'\.\..*\.env'
            ]
        }
    
    def scan_output(self, llm_output):
        """Scan LLM output for security threats"""
        threats = []
        
        for threat_type, patterns in self.dangerous_patterns.items():
            for pattern in patterns:
                matches = re.finditer(pattern, llm_output, re.IGNORECASE | re.MULTILINE)
                for match in matches:
                    threats.append({
                        'type': threat_type,
                        'pattern': pattern,
                        'match': match.group(),
                        'position': match.span(),
                        'severity': self.get_severity(threat_type)
                    })
        
        return threats
    
    def get_severity(self, threat_type):
        """Get severity level for threat type"""
        severity_map = {
            'xss': 'HIGH',
            'sql_injection': 'CRITICAL',
            'command_injection': 'CRITICAL',
            'path_traversal': 'HIGH'
        }
        return severity_map.get(threat_type, 'MEDIUM')
    
    def is_safe_output(self, llm_output):
        """Check if output is safe to use"""
        threats = self.scan_output(llm_output)
        critical_threats = [t for t in threats if t['severity'] == 'CRITICAL']
        return len(critical_threats) == 0, threats
```

```python
class ContentSecurityAnalyzer:
    def __init__(self):

    def analyze_content_safety(self, content):
        """Use ML to analyze content safety"""
        
        # Extract features
        features = self.extract_security_features(content)
        
        # Predict threat probability
        threat_probability = self.ml_model.predict_proba([features])[0]
        
        # Classify content
        is_malicious = threat_probability[1] > 0.7  # Threshold for malicious
        confidence = max(threat_probability)
        
        return {
            'is_malicious': is_malicious,
            'confidence': confidence,
            'threat_probability': threat_probability[1],
            'features': features
        }
    
    def extract_security_features(self, content):
        """Extract security-relevant features from content"""
        features = []
        
        # Length features
        features.append(len(content))
        features.append(len(content.split()))
        
        # Character distribution
        features.append(content.count('<'))
        features.append(content.count('>'))
        features.append(content.count(';'))
        features.append(content.count('\''))
        features.append(content.count('"'))
        
        # Keyword presence
        dangerous_keywords = [
            'script', 'eval', 'exec', 'system', 'drop', 'delete',
            'union', 'select', 'insert', 'update', 'create', 'alter'
        ]
        
        for keyword in dangerous_keywords:
            features.append(content.lower().count(keyword))
        
        # Entropy (randomness measure)
        features.append(self.calculate_entropy(content))
        
        return features
    
    def calculate_entropy(self, text):
        """Calculate Shannon entropy of text"""
        import math
        from collections import Counter
        
        if not text:
            return 0
        
        counter = Counter(text)
        length = len(text)
        entropy = 0
        
        for count in counter.values():
            probability = count / length
            entropy -= probability * math.log2(probability)
        
        return entropy
```

**1. Comprehensive Output Filtering**
```python
import html
import re
import urllib.parse

class LLMOutputSanitizer:
    def __init__(self):
        self.html_tags_whitelist = {
            'p', 'br', 'strong', 'em', 'u', 'ol', 'ul', 'li', 'h1', 'h2', 'h3'
        }
        
        self.dangerous_protocols = {
            'javascript:', 'data:', 'vbscript:', 'file:', 'ftp:'
        }
    
    def sanitize_for_html(self, content):
        """Sanitize content for safe HTML rendering"""
        
        # HTML encode all content first
        sanitized = html.escape(content)
        
        # Allow only whitelisted HTML tags
        sanitized = self.allow_safe_html_tags(sanitized)
        
        # Remove dangerous attributes
        sanitized = self.remove_dangerous_attributes(sanitized)
        
        # Validate URLs
        sanitized = self.sanitize_urls(sanitized)
        
        return sanitized
    
    def sanitize_for_sql(self, content):
        """Sanitize content for SQL usage"""
        
        # Remove SQL metacharacters
        dangerous_chars = ["'", '"', ';', '--', '/*', '*/', 'xp_', 'sp_']
        
        sanitized = content
        for char in dangerous_chars:
            sanitized = sanitized.replace(char, '')
        
        # Escape remaining single quotes
        sanitized = sanitized.replace("'", "''")
        
        # Limit length
        sanitized = sanitized[:1000]
        
        return sanitized
    
    def sanitize_for_command(self, content):
        """Sanitize content for command execution"""
        
        # Remove command injection characters
        dangerous_chars = [';', '|', '&', '$', '`', '(', ')', '<', '>', '\n', '\r']
        
        sanitized = content
        for char in dangerous_chars:
            sanitized = sanitized.replace(char, '')
        
        # Allow only alphanumeric and safe characters
        sanitized = re.sub(r'[^a-zA-Z0-9\s\-_.]', '', sanitized)
        
        return sanitized
    
    def sanitize_file_path(self, path):
        """Sanitize file path to prevent directory traversal"""
        
        # Remove path traversal sequences
        sanitized = path.replace('..', '')
        sanitized = sanitized.replace('//', '/')
        sanitized = sanitized.replace('\\\\', '\\')
        
        # Remove dangerous path components
        dangerous_paths = ['/etc/', '/proc/', '/sys/', 'C:\\Windows\\', 'C:\\System32\\']
        for dangerous in dangerous_paths:
            if dangerous.lower() in sanitized.lower():
                raise ValueError(f"Dangerous path detected: {dangerous}")
        
        # Ensure path is relative and within allowed directory
        sanitized = sanitized.lstrip('/')
        sanitized = sanitized.lstrip('\\')
        
        return sanitized
    
    def allow_safe_html_tags(self, content):
        """Allow only whitelisted HTML tags"""
        
        # Remove all HTML tags except whitelisted ones
        def replace_tag(match):
            tag = match.group(1).lower()
            if tag in self.html_tags_whitelist:
                return match.group(0)
            else:
                return ''
        
        # Remove opening tags not in whitelist
        content = re.sub(r'<(/?)(\w+)[^>]*>', replace_tag, content)
        
        return content
    
    def remove_dangerous_attributes(self, content):
        """Remove dangerous HTML attributes"""
        
        dangerous_attrs = [
            'onclick', 'onload', 'onerror', 'onmouseover', 'onfocus',
            'onblur', 'onchange', 'onsubmit', 'href', 'src'
        ]
        
        for attr in dangerous_attrs:
            content = re.sub(f'{attr}\\s*=\\s*["\'][^"\']*["\']', '', content, flags=re.IGNORECASE)
        
        return content
    
    def sanitize_urls(self, content):
        """Sanitize URLs to prevent malicious redirects"""
        
        url_pattern = r'https?://[^\s<>"\']+|www\.[^\s<>"\']+|[^\s<>"\']+\.[a-z]{2,}[^\s<>"\']*'
        
        def validate_url(match):
            url = match.group(0)
            
            # Check for dangerous protocols
            for protocol in self.dangerous_protocols:
                if url.lower().startswith(protocol):
                    return '[BLOCKED_URL]'
            
            # Validate domain (basic check)
            try:
                parsed = urllib.parse.urlparse(url if url.startswith('http') else f'http://{url}')
                if parsed.netloc:
                    return url
                else:
                    return '[INVALID_URL]'
            except:
                return '[INVALID_URL]'
        
        return re.sub(url_pattern, validate_url, content, flags=re.IGNORECASE)
```

**2. Context-Aware Sanitization**
```python
class ContextAwareSanitizer:
    def __init__(self):
        self.sanitizers = {
            'html': self.sanitize_html_context,
            'sql': self.sanitize_sql_context,
            'command': self.sanitize_command_context,
            'json': self.sanitize_json_context,
            'xml': self.sanitize_xml_context,
            'url': self.sanitize_url_context
        }
    
    def sanitize_by_context(self, content, context_type):
        """Sanitize content based on usage context"""
        
        if context_type not in self.sanitizers:
            raise ValueError(f"Unknown context type: {context_type}")
        
        return self.sanitizers[context_type](content)
    
    def sanitize_html_context(self, content):
        """Sanitize for HTML context"""
        # HTML encode and remove scripts
        sanitized = html.escape(content)
        sanitized = re.sub(r'<script[^>]*>.*?</script>', '', sanitized, flags=re.IGNORECASE | re.DOTALL)
        return sanitized
    
    def sanitize_sql_context(self, content):
        """Sanitize for SQL context"""
        # Use parameterized queries instead, but if needed:
        return content.replace("'", "''").replace('"', '""')
    
    def sanitize_command_context(self, content):
        """Sanitize for command execution context"""
        # Only allow alphanumeric and safe characters
        return re.sub(r'[^a-zA-Z0-9\s\-_.]', '', content)
    
    def sanitize_json_context(self, content):
        """Sanitize for JSON context"""
        import json
        try:
            # Validate JSON structure
            json.loads(content)
            return content
        except json.JSONDecodeError:
            # If not valid JSON, escape it
            return json.dumps(content)
    
    def sanitize_xml_context(self, content):
        """Sanitize for XML context"""
        # XML encode special characters
        xml_chars = {'&': '&amp;', '<': '&lt;', '>': '&gt;', '"': '&quot;', "'": '&apos;'}
        for char, encoded in xml_chars.items():
            content = content.replace(char, encoded)
        return content
    
    def sanitize_url_context(self, content):
        """Sanitize for URL context"""
        return urllib.parse.quote(content, safe='')
```

```python
import subprocess
import shlex

class SecureCommandExecution:
    def __init__(self):
        self.allowed_commands = {
            'ls', 'cat', 'grep', 'find', 'wc', 'sort', 'head', 'tail'
        }
        self.allowed_directories = {
            '/tmp/safe_area', '/home/user/documents'
        }
    
    def execute_llm_suggested_command(self, llm_output):
        """Safely execute LLM-suggested commands"""
        
        # Parse command from LLM output
        command_parts = self.parse_command(llm_output)
        
        if not command_parts:
            raise ValueError("No valid command found in LLM output")
        
        # Validate command
        self.validate_command(command_parts)
        
        # Execute safely
        return self.safe_execute(command_parts)
    
    def parse_command(self, llm_output):
        """Parse and extract command from LLM output"""
        
        # Look for command patterns
        command_patterns = [
            r'run:\s*(.+)',
            r'execute:\s*(.+)',
            r'command:\s*(.+)',
            r'`([^`]+)`'
        ]
        
        for pattern in command_patterns:
            match = re.search(pattern, llm_output, re.IGNORECASE)
            if match:
                command = match.group(1).strip()
                return shlex.split(command)
        
        return None
    
    def validate_command(self, command_parts):
        """Validate command for security"""
        
        if not command_parts:
            raise ValueError("Empty command")
        
        # Check if base command is allowed
        base_command = command_parts[0]
        if base_command not in self.allowed_commands:
            raise ValueError(f"Command not allowed: {base_command}")
        
        # Validate arguments
        for arg in command_parts[1:]:
            if not self.is_safe_argument(arg):
                raise ValueError(f"Unsafe argument: {arg}")
    
    def is_safe_argument(self, arg):
        """Check if command argument is safe"""
        
        # Check for dangerous characters
        dangerous_chars = [';', '|', '&', '$', '`', '(', ')', '<', '>']
        if any(char in arg for char in dangerous_chars):
            return False
        
        # Check for path traversal
        if '..' in arg:
            return False
        
        # If it's a file path, ensure it's in allowed directory
        if arg.startswith('/'):
            return any(arg.startswith(allowed) for allowed in self.allowed_directories)
        
        return True
    
    def safe_execute(self, command_parts):
        """Execute command safely"""
        
        try:
            # Execute with restricted environment
            result = subprocess.run(
                command_parts,
                capture_output=True,
                text=True,
                timeout=30,  # Prevent long-running commands
                cwd='/tmp/safe_area',  # Restrict working directory
                env={'PATH': '/usr/bin:/bin'}  # Minimal environment
            )
            
            if result.returncode == 0:
                return result.stdout
            else:
                raise RuntimeError(f"Command failed: {result.stderr}")
                
        except subprocess.TimeoutExpired:
            raise RuntimeError("Command timed out")
        except Exception as e:
            raise RuntimeError(f"Command execution failed: {e}")
```

```python
        self.validate_api_parameters(request['params'])
        
        # Check rate limits
        if not self.rate_limiter.check_limit():
            raise ValueError("Rate limit exceeded")
    
    def validate_api_parameters(self, params):
        """Validate API parameters"""
        
        # Check parameter names
        allowed_params = {
            'query', 'limit', 'offset', 'sort', 'filter', 'user_id'
        }
        
        for param_name in params.keys():
            if param_name not in allowed_params:
                raise ValueError(f"Parameter not allowed: {param_name}")
        
        # Validate parameter values
        for param_name, param_value in params.items():
            if not self.is_safe_parameter_value(param_value):
                raise ValueError(f"Unsafe parameter value: {param_name}")
    
    def is_safe_parameter_value(self, value):
        """Check if parameter value is safe"""
        
        # Convert to string for validation
        str_value = str(value)
        
        # Check length
        if len(str_value) > 1000:
            return False
        
        # Check for injection patterns
        dangerous_patterns = [
            r'<script', r'javascript:', r'data:', r'vbscript:',
            r'union\s+select', r'drop\s+table', r'delete\s+from'
        ]
        
        for pattern in dangerous_patterns:
            if re.search(pattern, str_value, re.IGNORECASE):
                return False
        
        return True
    
    def safe_api_call(self, request):
        """Make safe API call"""
        
        url = f"{self.base_url}{request['endpoint']}"
        headers = {
            'Authorization': f'Bearer {self.api_key}',
            'Content-Type': 'application/json'
        }
        
        try:
            if request['method'] == 'GET':
                response = requests.get(
                    url, 
                    params=request['params'], 
                    headers=headers,
                    timeout=30
                )
            else:
                response = requests.post(
                    url, 
                    json=request['params'], 
                    headers=headers,
                    timeout=30
                )
            
            response.raise_for_status()
            return response.json()
            
        except requests.RequestException as e:
            raise RuntimeError(f"API call failed: {e}")
```

### Output Security Scorecard
```python
class OutputSecurityScorecard:
    def __init__(self):
        self.metrics = {
            'total_outputs': 0,
            'sanitized_outputs': 0,
            'blocked_outputs': 0,
            'vulnerability_attempts': 0,
            'successful_attacks': 0
        }
    
    def record_output_processing(self, output, was_sanitized, was_blocked, contained_threats):
        """Record output processing metrics"""
        
        self.metrics['total_outputs'] += 1
        
        if was_sanitized:
            self.metrics['sanitized_outputs'] += 1
        
        if was_blocked:
            self.metrics['blocked_outputs'] += 1
        
        if contained_threats:
            self.metrics['vulnerability_attempts'] += 1
    
    def calculate_security_score(self):
        """Calculate overall output security score"""
        
        if self.metrics['total_outputs'] == 0:
            return 100.0
        
        # Calculate sanitization rate
        sanitization_rate = (
            self.metrics['sanitized_outputs'] / 
            self.metrics['total_outputs']
        ) * 100
        
        # Calculate threat detection rate
        if self.metrics['vulnerability_attempts'] > 0:
            detection_rate = (
                self.metrics['blocked_outputs'] / 
                self.metrics['vulnerability_attempts']
            ) * 100
        else:
            detection_rate = 100.0
        
        # Calculate attack success rate (lower is better)
        if self.metrics['vulnerability_attempts'] > 0:
            attack_success_rate = (
                self.metrics['successful_attacks'] / 
                self.metrics['vulnerability_attempts']
            ) * 100
        else:
            attack_success_rate = 0.0
        
        # Weighted security score
        security_score = (
            sanitization_rate * 0.3 +
            detection_rate * 0.5 +
            (100 - attack_success_rate) * 0.2
        )
        
        return min(100.0, max(0.0, security_score))
```
